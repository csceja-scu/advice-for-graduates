# Advice for Graduates — Entering the Workforce

An interactive, single-file HTML slide deck created for the Santa Clara
University Alumni Association (COMM 129: Advanced Public Speaking). A
peer-to-peer guide for the Class of 2026 covering three topics:

1. **AI as Augmentation, not Replacement** — using AI without losing your voice
2. **Self Care & Boundaries** — managing work / life balance from day one
3. **Workplace Communication & Conflict** — navigating difficult conversations

## Viewing it

Open `advice-for-graduates.html` in any modern browser. The file is fully
self-contained (all images and styles embedded) and works offline.

## Controls

- **Arrow keys** (← / →), Space, or PageUp / PageDown — move between slides
- **Number keys** (1–9) — jump to a slide
- **Home / End** — first / last slide
- **F** — fullscreen
- On-screen prev/next buttons, a dot strip, and clickable agenda rows
- `?slide=N` in the URL opens directly to a slide (shareable deep links)

## Design

A "Spectrum" system: one full-bleed color gradient per section, set in
Helvetica Neue, with display headings on the gradient and body copy on
frosted-glass cards. All text/background pairings meet WCAG AA contrast.
